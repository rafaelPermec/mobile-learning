## Projetos do Workspace

Este workspace reúne dois projetos mobile voltados para Pokédex, cada um com uma abordagem diferente de implementação e interface.

## Flutter

[flutter/pokedex_app_flutter](flutter/pokedex_app_flutter) é um aplicativo Flutter em Dart com foco em uma tela de detalhes de movimento Pokémon. A aplicação usa Material Design, tema personalizado com identidade Pokédex e uma interface simples, estática e responsiva.

## React Native

[react_native/PokedexApp](react_native/PokedexApp) é um aplicativo React Native em TypeScript, estruturado para uma experiência mobile multiplataforma. O projeto já inclui organização inicial em componentes e telas, além da configuração típica de build, testes e navegação do ecossistema React Native.

## Resumo Rápido

- Flutter: interface em Dart com uma tela de detalhes pronta para evoluir.
- React Native: base em TypeScript para ampliar a Pokédex com componentes e telas separadas.
